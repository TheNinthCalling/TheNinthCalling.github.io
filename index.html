Projects

Carbon Flights (October 2023)
Environmentally-Conscious Flight Search Platform

An innovative web platform designed to bridge the gap between budget-friendly travel and environmental responsibility. Carbon Flights offers travelers a unique experience with its interactive globe homepage. As users spin the globe and select destinations, the platform provides eco-friendly flight options. While the integration with Google APIs to pinpoint exact coordinates and nearby airports is still in progress, the platform's backend, built with Python and Flask, seamlessly interfaces with a MongoDB database.

Live Demo | GitHub Repository

Video Game Store Management System (August 2023)
Comprehensive Digital Game Store Solution

This project encapsulates a holistic approach to digital game store management. Engineered using Java Spring Boot and following MVC principles, the system offers robust data management through MySQL, Docker, GraphQL, and REST APIs. Enhanced security features, including OAuth-based user authentication, ensure safe and smooth transactions. The system also boasts features for inventory management and user profile CRUD operations.

GitHub Repository

Full-Stack Mock HaaS Site (Spring 2023)
Innovative Hardware-as-a-Service Platform

Pioneering the concept of temporary hardware acquisition for project ideation, this mock HaaS platform offers users a dynamic and responsive experience. The front-end, crafted using React, is backed by a Python-based backend ensuring smooth data operations. Deployed on Heroku, the platform guarantees consistent uptime and performance.
